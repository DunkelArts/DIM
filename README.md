# Destiny 2 Inventory Manager - Desktop Version

## Install

Just run the EXE to install locally.

## Build yourself

To develop you need to have Node.js installed

[Get Node.js](https://nodejs.org/en/)

```bash
# install all dependencies
npm install

# run with node
npm start

# create .exe
npm run build
```
